BZOJ4292 Rownanie¶

Description¶

对于一个正整数 $n$，定义 $f(n)$ 为它十进制下每一位数字的平方的和。 现在给定三个正整数 $k$, $a$, b，请求出满足 $a\leq n\leq b$ 且 $k\times f(n) = n$ 的 $n$ 的个数。

Solution¶

$k$ 是给定的，那么对于一个 $f(n)$，只至多一个 $n$ 是合法的。我们直接枚举 $f(n)$ 即可。

Code¶

namespace solve
{
    void main()
    {
        long long k, a, b;
        cin >> k >> a >> b;
        long long ans = 0;
        for (int i = 1; i <= 1500; i++)
        {
            __int128_t x = __int128_t(k) * i;
            if (x >= a && x <= b)
            {
                int j = 0;
                while (x)
                    j += (x % 10) * (x % 10), x /= 10;
                if (j == i)
                    ans++;
            }
        }
        cout << ans << endl;
    }
}
