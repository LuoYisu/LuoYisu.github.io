一个奇妙的建模题。

Description

给出数轴上三个棋子 ，你可以对棋子进行如下操作：

  • 将一个棋子跳到另一个棋子的另一侧，距离不变，而且不能同时跳过两个棋子。

问最少花费多少步能让三个棋子的位置变为 。

。

Solution

看到这个题感觉非常神秘，手玩一下一个点的所有变化可以发现每个状态最多只有三种变化，如果距离两两相同就只有两种。

然后发现其中一种变化相当于是倒退，而如果距离两两相同就没法倒退了。

我们把距离两两相同的状态看作根，这样所有变化就变成了一棵二叉树。

题目要求的就是两个状态在二叉树上的距离。

再深究一下，往根的方向走会让状态的距离之和变小（例如 变成 ），我们有一个很显然的暴力就是对初始状态和结束状态分别走到根，然后看什么时候相交，这是 的（ 是值域）。

再观察一下变化树，发现往根的方向走一步相当于对距离执行一次辗转相减，那么我们用辗转相除就可以 向上跳了。

这样我们已经可以快速跳，并求出深度了，下一步是求 LCA，树剖之类的方法显然不好用，我们二分一下 LCA 就好了，复杂度 。
